# `@peculiar/pdf-cli`

Command-line interface for PDF document manipulation and analysis.

## Installation

```bash
npm install -g @peculiar/pdf-cli
```

## Commands

### info

Displays detailed information about a PDF document.

```bash
pvpdf info <pdf-file>
```

Example:

```bash
pvpdf info document.pdf
```
